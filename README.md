# Rexar - Cybersecurity Solutions

A modern, lightweight cybersecurity company website built with React.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Modern black & white minimalist design
- Lightweight (minimal dependencies)
- Responsive navigation with dropdown menus
- Fast loading times
- React Router for navigation

## Tech Stack

- React 18
- Vite (fast build tool)
- React Router DOM
- Pure CSS (no heavy UI libraries)
